# Quickstart: Garden Room Structural Logic

## Prerequisites

- Node.js 18+
- npm 9+

## Install

1. Install dependencies:
   - `npm install`

## Run

1. Start the Angular app:
   - `npm start`
2. Open http://localhost:4200

## Feature Entry

- Planned route: `/garden-room`
- The feature will be accessible from the project dashboard once routes are
  wired.

## Smoke Test

1. Enter global height limit and offsets.
2. Confirm derived front/back wall heights update immediately.
3. Add a wall and verify stud positions and cut list output.
